# asdf-lazysql plugin for the  [asdf version manager](https://asdf-vm.com/)

asdf-lazysql is a small C utility that lets you attach and detach terminal sessions, similar to the detach feature of screen/tmux — but without the terminal emulation, multiple windows, or other overhead.
https://github.com/jorgerojas26/lazysql

## Install plugin

```bash
asdf plugin add lazysql https://github.com/a1xs/asdf_lazysql
```
#  Install app

## Install latest version
```bash
asdf install lazysql latest
```

## Install specific version
```bash
asdf install lazysql 0.4.3
```
